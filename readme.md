CheaterBot is currently in an early testing phase. Users should expect frequent updates and potential bugs as the software is still under active development.

Language Support
Currently, CheaterBot is only available in English.

However, we are planning to implement Spanish support in the near future. Depending on development progress and user demand, support for additional languages may also be added at a later stage.

Installation and Execution

python main.py
